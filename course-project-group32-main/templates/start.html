Welcome to ISS, a modern web-based video editor

About

Our video editor, built using Flask, HTML, JavaScript, CSS, CockroachDB, and Microsoft Azure, provides users with a comprehensive set of tools for editing videos seamlessly in the browser.

Features

  • Intuitive user interface
  • Real-time collaboration
  • Advanced editing tools
  • Integration with cloud storage
  • Scalable infrastructure

Technology Stack

  • Flask for backend development
  • HTML, JavaScript, and CSS for frontend
  • CockroachDB for distributed database management
  • Microsoft Azure for cloud hosting and services
